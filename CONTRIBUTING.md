# Contributing to plugin

## Set up

Install dev deps after git clone the repo.

```bash
# npm is not allowed.
$ pnpm i
```

## Build

Transform with babel and rollup.

```bash
$ pnpm run build

# Build and monitor file changes
$ pnpm run dev

```

## Dev Plugin

```bash
# This Step must only be executed in Build
$ pnpm dev
```
